# ERCOT Transmission Cost Allocation Study

An open-source methodology lab for comparing incumbent and alternative ERCOT transmission cost allocation designs, including 4CP, net-load variants, annual coincident peak variants, hybrid volumetric/peak designs, and congestion-based methods.

## Core research question
Which ERCOT transmission cost allocation methodology best balances cost causation, fairness, revenue stability, administrative implementability, and resistance to strategic gaming?

## Comparison set
1. ERCOT 4CP
2. 4CP Net Load
3. 12CP
4. 12CP Net Load
5. Volumetric + 12CP Net Load hybrid
6. CBTCA
7. CBTCA sensitivity variants

## v1 granularity
- Zone
- TDSP
- Proxy class, large-load vs retail

## Important data note
This repository must not publish raw licensed data from Yes Energy or Enverus. Public artifacts include code, schemas, methodology docs, synthetic fixtures, validation notebooks, and public-safe examples only.
