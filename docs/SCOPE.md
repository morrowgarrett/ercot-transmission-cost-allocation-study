# Scope

## Core research question
Which ERCOT transmission cost allocation methodology best balances cost causation, fairness, revenue stability, administrative implementability, and resistance to strategic gaming?

## Objective
Extend Garrett's existing CBTCA work into a full ERCOT transmission cost allocation methodology lab built for public scrutiny, reproducibility, and adversarial review.

## Comparison set
1. ERCOT 4CP
2. 4CP Net Load
3. 12CP
4. 12CP Net Load
5. Volumetric + 12CP Net Load hybrid
6. CBTCA
7. CBTCA sensitivity variants

## Evaluation criteria
1. Cost causation fidelity
2. Gaming resistance
3. Revenue sufficiency / stability
4. Administrative implementability
5. Distributional equity

## v1 granularity
- Zone
- TDSP
- Proxy class, large-load vs retail

## Study years
Preferred: 2023, 2024, 2025
Fallback: 2024-only for first integrated run if coverage is incomplete.

## Public/private data boundary
Public outputs may include code, schemas, docs, tests, synthetic fixtures, and public-safe notebooks.
Licensed Yes Energy and Enverus raw data must remain local.
