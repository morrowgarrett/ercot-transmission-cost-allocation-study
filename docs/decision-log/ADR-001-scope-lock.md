# ADR-001 Scope Lock

- Status: Accepted
- Date: 2026-04-11

## Context
The project began as a CBTCA-versus-4CP comparison but is being expanded into a public methodology lab. Without an explicit scope lock, the work will drift toward customer-level claims, extra methodologies, and premature implementation complexity.

## Decision
Lock the study to seven methodologies, five evaluation criteria, v1 reporting at zone + TDSP + proxy class, and preferred study years 2023-2025.

## Alternatives considered
- Keep scope open and add methodologies ad hoc
- Target customer-level reconstruction in v1
- Limit the work to CBTCA versus incumbent 4CP only

## Consequences
This prevents uncontrolled expansion, keeps the public claims defensible, and gives the implementation a stable target. Some attractive extensions are deferred to later versions.
