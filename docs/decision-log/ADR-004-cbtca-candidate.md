# ADR-004 CBTCA as Candidate

- Status: Accepted
- Date: 2026-04-11

## Context
Garrett already has substantial intellectual investment in CBTCA. That creates a strong temptation to structure the project as a proof of superiority rather than an honest lab.

## Decision
Treat CBTCA as one candidate methodology under the same scorecard and confidence framework as the incumbent and hybrid alternatives.

## Alternatives considered
- Build the project as an advocacy package for CBTCA
- Use 4CP only as a strawman baseline
- Skip hybrid reform candidates

## Consequences
The work becomes more credible to external reviewers. It also creates the possibility that a different methodology may emerge as more adoptable than CBTCA.
